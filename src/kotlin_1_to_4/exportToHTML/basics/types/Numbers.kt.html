28    
29

33    
34       // "Widening" example 
35       // Widening refers to casting a type to it's "super" type, in the context of numbers, a number that can 
36       // handle more bits, or i.e. width 
37       var wideningExample: Long = myInt.toLong() 
38    
39    
40       // When it goes the other way around it's the same deal. We must "cast" explicitly using the .to* keyword
