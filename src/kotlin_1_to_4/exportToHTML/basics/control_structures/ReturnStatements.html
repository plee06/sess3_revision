ReturnStatements 
1    In Kotlin the return statement is similar to Java. 
2     
3     
4    The thing with Kotlin though is that because we can nest functions in a "functional programming" way 
5    the return will return out of the inner most function, i.e. the scope it is currently in. 
6     
7     
8    Also, when it comes to closures like running a .forEach and testing for some condition, the return can have a label 
9    with return@forEach or we can label the .forEach stop@ 
10    
11   Refer to the documentation or Programming Kotlin book for more details.
